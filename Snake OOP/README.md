# Snake OOP

Object-Oriented Programming version of the classic Snake game.

## Features
- One player mode
- Two players mode (same field / split screen)
- Three difficulty levels
- Records saving
- Pause and exit

## Controls
- Player 1: WASD
- Player 2: Arrows
- P: Pause
- X: Exit

## How to build
- For Fi.cpp: `g++ Fi.cpp -o snake.exe`
- For Snake OOP.cpp: `g++ "Snake OOP.cpp" -o snake_oop.exe`
